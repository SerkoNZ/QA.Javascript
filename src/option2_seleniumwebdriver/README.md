# Instructions
## Pre-requisites
* node 12+
* python3

## Running demo app locally
* `cd src/demoApp/`
* `npm install`
* `npm run dev`
* open `http://localhost:3000` in your browser

## To run tests locally:
1. `npm install`
2. `npm run test
